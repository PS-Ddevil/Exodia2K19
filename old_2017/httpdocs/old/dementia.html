Dementia will be an online programming competition.

Description and Rules :

1. Competition is for individuals. No teams are allowed.

2. There is no restriction on the programming language to be used.

3. The contest is open only to students, no professionals are allowed.

4. You will receive one point for solving a problem (all test-cases passed, no partial credit), regardless of the difficulty level of the problem.

5. Users are ranked by the number of problems solved. Ties will be broken by the total time taken, in ascending order of time.

6. The total time is the sum of the time consumed for each problem solved. The time consumed for a solved problem is the time elapsed from the beginning of the contest to the submission of the first accepted run plus 20 minutes for every previously rejected run for that problem. There's no time consumed for a problem that is not solved.
